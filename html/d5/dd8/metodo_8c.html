Carrega as classes a partir do arquivo ".class" chamando o leitor e em seguida carrega essa classe para a memoria. Alem disso, confere se a classe ja esta carregada antes de chamar o Leitor. Tambem retorna um ponteiro para uma classe no array ou o seu nome, pelas funcoes implementadas.
